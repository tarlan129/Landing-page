# The Odin Project — Landing Page

Second project from The Odin Project Foundations course. Built a full landing page from scratch using HTML and CSS.

## What I Learned

This project pushed me further than the first one. On top of what I already knew, I worked with:

- Flexbox — nested containers, `flex-direction`, `justify-content`, `align-items`
- Styling `<a>` tags as buttons using `inline-block`, `border`, and `padding`
- `transition` for smooth hover effects
- `span` for inline color changes without breaking layout
- `border-radius` for polishing visuals
- `box-sizing: border-box` reset
- `line-height` for readable text
- Building consistent UI with a fixed color palette
- I tried to use everything I learned throughout the course in one project.

The layout part was manageable but getting Flexbox to behave exactly as intended — especially nested flex containers — took real effort.
